# 🛕 ISKCON Project

A full-stack web application designed to provide users with a centralized platform for accessing ISKCON-related information such as temples, events, and spiritual resources.

The project focuses on clean architecture, secure authentication, and a smooth user experience.

---

## 🌟 Features

- 🔐 Secure User Authentication (JWT-based)
- 🛕 Temple Information Management
- 📅 Event Listing and Tracking
- 📿 Spiritual Content Access
- 📱 Fully Responsive UI
- ⚡ Fast and optimized backend APIs

---

## 🧠 Problem It Solves

People often struggle to find **organized spiritual content and temple/event information in one place**.

This platform solves that by:
- Providing centralized access to ISKCON resources  
- Simplifying navigation and discovery  
- Offering a clean and user-friendly interface
